# HRDWR
HRDWR (pronounced hardware, because who needs vowels anyways?) is a Go library for fetching health data on the local systems hardware.

## Features
* CPU usage
* Memory usage
* Disk space used
* Temperature sensor data
* Fan sensor data
* Voltage sensor data

## Install 
```bash
$ go get github.com/axnion/hrdwr
```
